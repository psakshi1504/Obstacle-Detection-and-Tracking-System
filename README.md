# 🚀 Obstacle Detection and Tracking System

## 📌 Overview
This project is an Arduino Uno-based obstacle detection and tracking system using an ultrasonic sensor and servo motor. It detects objects, tracks them, and displays distance on an LCD.

## 🔧 Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Buzzer
- LEDs

## ⚙️ Features
- Real-time obstacle detection
- Servo-based scanning (radar-like)
- LED and buzzer alert system
